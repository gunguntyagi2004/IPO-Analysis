## Overview
A complete end-to-end data analysis of **652 Indian IPOs** from January 2010 to May 2026. This project uncovers patterns in listing performance, subscription trends, and long-term wealth creation in the Indian primary capital market.

## Key Findings
- 57.4% of IPOs gave positive listing day returns
- Total subscription has **r = 0.71** correlation with listing gain
- **2020** was the best year — avg listing gain of 38.7%
- **2025** hit a record **107 IPOs** listed
- HNI investors average **71x** subscription vs 16x retail
- ~30% of IPOs are long-term wealth creators (>100% current gain)
